Add the current weather from The Netherlands in your posts

== Description ==

Add the current weather from The Netherlands in your posts, pages or add a widget, this is only interesting for people that live in the Netherlands, Belgium or Germany.

Related Links:

* <a href="http://kierownik.nl/wordpress-buienradar-plugin-and-widget" title="Plugin home">kierownik.nl</a>
* <a href="http://http://gratisweerdata.buienradar.nl" title="gratis weer data">buienradar.nl</a>

== Installation ==

1. Upload folder `buienradar` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Then head over to "settings" => "buienradar.nl"
